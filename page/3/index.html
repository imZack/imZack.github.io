Rsync 更改預設 SSH Port

發表於 12月 1 2014   |  

一般 Rsync 指令如下

rsync -av /home/zack username@host

若要更改 SSH PORT (default 22) 的話則可下

rsync -av -e 'ssh -p 2222' /home/zack username@host
